# 湖北高考志愿助手（样本版）

直接双击 `index.html` 即可使用，无需安装依赖或启动服务器。

## 当前能力

- 输入湖北高考分数、首选科目和省内位次。
- 按目标高校省份筛选代表性985、211/双一流和传统重点本科。
- 使用2025年湖北普通批投档线和一分一段表进行位次匹配。
- 输出冲、稳、保档位，并按就业目标重新排序专业。
- 标记未来十年重点方向、条件型专业和高风险专业。

## 数据边界

这是公开历史样本演示，不是全国完整数据库，也不提供录取保证。专业标签为代表性方向，不等同于2026年实际招生专业。正式填报前必须核对：

1. 湖北省教育考试院发布的2026招生计划和投档规则；
2. 高校2026本科招生章程；
3. 专业组内全部专业、调剂范围、体检及单科限制；
4. 公费师范、优师专项等提前批项目的履约政策。

## 文件结构

- `index.html`：页面结构
- `styles.css`：响应式视觉样式
- `app.js`：位次匹配、筛选排序与渲染
- `data/admissions-hubei.js`：湖北历史投档样本
- `data/majors.js`：专业就业与风险规则
- `data/sources.js`：公开数据来源
