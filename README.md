# 2026全国高考志愿助手

直接双击 `index.html` 即可使用，无需安装依赖或启动服务器。

## 当前能力

- 考生省份覆盖全国31个省级地区，并根据当地模式显示`3+1+2`、`3+3`或传统文理科目。
- 只输入2026高考分数，系统优先用2026一分一段显示累计位次；缺失时回退到2025历史数据并明确标注。
- 目标省份严格参与筛选，结果只包含所选省份的院校。
- 默认按普通批最低投档分从高到低排列。
- 内置教育部截至2026年6月17日公布的1412所本科学校元数据。
- 输出冲、稳、保档位，并按就业目标重新排序专业。
- 标记未来十年重点方向、条件型专业和高风险专业。

## 当前数据进度

- 全国31个省级地区均可生成普通类本科建议，覆盖`3+1+2`、`3+3`和传统文理科。
- 湖北优先使用本地接入的2026官方普通类一分一段数据和2025普通本科批投档样本。
- 其他省份通过公开聚合接口优先查询2026一分一段；2026缺失时自动采用该省2025数据，并显示“历史数据参考”。
- 若一分一段或历史投档接口暂不可得，系统退回分数参考或教育部本科院校目录，标为“参考”，不虚构位次和录取概率。

## 数据边界

程序不提供录取保证。专业标签为代表性方向，不等同于2026年实际招生专业。正式填报前必须核对：

1. 考生所在省教育考试机构发布的2026招生计划和投档规则；
2. 高校2026本科招生章程；
3. 专业组内全部专业、调剂范围、体检及单科限制；
4. 公费师范、优师专项等提前批项目的履约政策。

## 文件结构

- `index.html`：页面结构
- `styles.css`：响应式视觉样式
- `app.js`：位次匹配、筛选排序与渲染
- `data/provinces.js`：全国省份、考试模式和选科配置
- `data/schools.js`：教育部2026全国本科院校元数据
- `data/admissions-hubei.js`：湖北历史投档样本
- `data/rank/2026/hubei.js`：湖北2026官方一分一段数据
- `data/majors.js`：专业就业与风险规则
- `data/sources.js`：公开数据来源
